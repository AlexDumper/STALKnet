# STALKnet Console Client

Консольный TUI клиент для STALKnet.

## Требования

- Go 1.21 или выше
- Терминал с поддержкой Unicode

## Установка

1. Установите Go: https://go.dev/dl/

2. Перейдите в директорию клиента:
```bash
cd client
```

3. Загрузите зависимости:
```bash
go mod tidy
```

4. Запустите клиент:
```bash
go run .
```

Или с именем пользователя:
```bash
STALKNET_USER=alice go run .
```

## Управление

| Клавиша | Действие |
|---------|----------|
| `Enter` | Отправить сообщение/команду |
| `Ctrl+C` | Выход |
| `Ctrl+L` | Очистить экран |
| `PgUp` | Прокрутка вверх |
| `PgDn` | Прокрутка вниз |

## Команды

| Команда | Описание |
|---------|----------|
| `/help` | Показать список команд |
| `/clear` | Очистить экран сообщений |
| `/nick <name>` | Сменить имя пользователя |
| `/connect` | Показать статус подключения |
| `/quit` | Выйти из приложения |
| `/mock <text>` | Отправить тестовое сообщение |
| `/mockmsg` | Сгенерировать случайное сообщение |
| `/mocktask` | Сгенерировать уведомление о задаче |
| `/scroll up/down` | Прокрутка сообщений |
